# API Testing Documentation

This guide provides instructions for testing the Gemini PDF Reader API using Postman or cURL.

## Server Details

- **Base URL**: `http://localhost:5050`
- **Port**: `5050`

## Endpoints

### 1. Upload and Analyze PDF

Processes a PDF file and returns an AI-generated response along with token usage metrics.

- **URL**: `/upload`
- **Method**: `POST`
- **Content-Type**: `multipart/form-data`

#### Request Body (form-data)

| Key      | Type | Description                                                                        |
| :------- | :--- | :--------------------------------------------------------------------------------- |
| `pdf`    | File | The PDF file to be analyzed (Required, max 10MB)                                   |
| `prompt` | Text | The instructions for Gemini (Optional, default: "Please summarize this PDF file.") |

#### Sample cURL Request

```bash
curl -X POST http://localhost:5050/upload \
  -F "pdf=@/path/to/your/file.pdf" \
  -F "prompt=Summarize this document"
```

#### Successful Response (200 OK)

```json
{
  "text": "The summary of the document content...",
  "usage": {
    "promptTokenCount": 125,
    "candidatesTokenCount": 45,
    "totalTokenCount": 170,
    "explanation": {
      "promptTokenCount": "Input tokens: These are the tokens in your prompt and the uploaded file...",
      "candidatesTokenCount": "Output tokens: These are the tokens generated by Gemini in its response.",
      "totalTokenCount": "Total tokens: The sum of input and output tokens..."
    }
  }
}
```

#### Error Response (400 Bad Request)

- Returned when no file is uploaded or a non-PDF file is provided.

```json
{ "error": "No file uploaded" }
```

#### Error Response (500 Internal Server Error)

- Returned if the Gemini API fails or there's a server-side issue.

```json
{ "error": "Unexpected error message" }
```

## Postman Instructions

1. Create a new request in Postman.
2. Set the method to **POST** and the URL to `http://localhost:5050/upload`.
3. Go to the **Body** tab and select **form-data**.
4. Add a key named `pdf`, set its type to **File**, and select your PDF.
5. (Optional) Add a key named `prompt`, set its type to **Text**, and enter your instructions.
6. Click **Send**.
